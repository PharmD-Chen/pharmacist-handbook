## 技术方法

### 网页解析
- 使用 `requests` 库获取网页内容
- 使用 `BeautifulSoup` 解析HTML
- **关键发现**: 药品信息位于页面的第一个 `<table>` 元素中

### 数据提取
- 使用正则表达式从文本流中提取各字段
- 字段顺序: 适应证 → 药理作用 → 用法用量 → 禁忌 → 不良反应 → 药物相互作用 → 注意事项 → 贮藏
- 使用非贪婪匹配 `(.+?)` 精确定位字段边界

### 文本处理
- 合并多个空白字符
- 替换特殊空格字符 (`\xa0`, `\u3000`)
- 精简版字段限制在300字符以内

# 药品手册 - GitHub Pages 部署包

## 📦 部署包内容

```
deploy/
├── index.html              # 主页面（已集成权限控制）
├── login.html              # 登录页面
├── auth.js                 # 认证和权限控制模块
├── data/
│   └── drugs/
│       ├── drugs.js        # 药品数据（1032个药品）
│       └── index.json      # 药品索引
├── DEPLOY.md               # 详细部署指南
└── README.md               # 本文件
```

## 🚀 快速部署

### 1. 本地测试

```bash
cd deploy
python3 -m http.server 8080
```

访问: http://localhost:8080/login.html

### 2. GitHub Pages 部署

1. 在 GitHub 创建新仓库
2. 将 `deploy/` 目录内容上传到仓库
3. 启用 GitHub Pages（Settings → Pages）
4. 访问: `https://YOUR_USERNAME.github.io/REPO_NAME/`

详细步骤见 [DEPLOY.md](DEPLOY.md)

## 🔐 安全提示

⚠️ **部署后请立即修改默认密码！**

修改方法：
1. 编辑 `auth.js` 文件
2. 找到 `USER_CONFIG` 对象
3. 修改各用户类型的 `password` 字段
4. 重新提交到 GitHub

- **indications / full_indications**: 功能主治（使用"功能主治"提取）

- **url.last_updated**: 2026-03-21

## 技术细节

### 中成药与西药字段差异

| 字段 | 西药 | 中成药 |
|------|------|--------|
| 适应症 | 适应证 | **功能主治** |
| 药理作用 | 药理作用 | **成份/药理作用** |
| 禁忌 | 禁忌 | **禁忌症** |

### 提取策略

```python
# 1. 功能主治（优先）
match = re.search(r'功能主治(.+?)(?:成份|药理作用|用法用量|...)', text)

# 2. 成份/药理作用
match = re.search(r'成份/药理作用(.+?)(?:用法用量|...)', text)

# 3. 禁忌症
match = re.search(r'禁忌症?(.+?)(?:不良反应|...)', text)
```

### 数据清理

- 移除了"剂型与规格"、"哪儿有"等混入内容
- 分割了adverse_reactions中混入的禁忌症内容
- 精简了interactions和precautions中的冗余信息

## 问题分类统计

### 主要问题类型

#### 1. 缺少数据来源网址 (544个药品)

**影响**: 无法追溯数据来源，无法验证内容真实性

**示例药品**:
- ID 1: 阿司匹林
- ID 7: 氟尿嘧啶
- ID 8: 盐酸贝尼地平
- ID 14: 注射用头孢西丁钠/氯化钠
- ID 21: 依洛尤单抗

**解决方案**: 
- 在湖南药事服务网搜索这些药品
- 补充 `url.hnysfww` 字段
- 记录到 `已补充药品网址.txt`

#### 2. 未标注数据来源 (134个药品)

**影响**: 无法确认数据来源的可靠性

**解决方案**:
- 追溯这些药品的数据来源
- 补充 `source` 字段
- 如无法确认来源，需要重新抓取

#### 3. pharmacokinetics字段问题 (部分药品)

**问题描述**: 部分药品的pharmacokinetics字段包含药效学描述而非药动学参数

**示例**:
```
❌ 错误: "本品具有抗炎作用，能抑制前列腺素合成..."（药效学）
✅ 正确: "口服后2小时达峰，半衰期8小时，经肝脏代谢..."（药动学）
```

**解决方案**:
- 从网站重新提取药动学参数
- 保留：达峰时间、半衰期、生物利用度、代谢途径等
- 移除：作用机制、治疗效果等药效学描述
